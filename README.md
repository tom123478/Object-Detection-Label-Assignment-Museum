# Object-Detection-Label-Assignment-Museum
# 目标检测标签分配博物馆

## 项目简介

本项目旨在创建一个"目标检测标签分配博物馆",展示和比较各种目标检测算法中使用的标签分配策略。
通过这个项目,我们希望能够帮助研究人员和开发者更好地理解不同标签分配方法的优缺点,并为选择合适的算法提供参考。

## 主要特点

- 收集并展示多种目标检测算法的标签分配策略
- 提供每种策略的详细说明和可视化展示
- 对比不同策略的优缺点和适用场景
- 包含代码示例和实现细节

## 包含的标签分配策略

1. 基于IoU的分配 (如YOLO, SSD)
2. 基于中心点的分配 (如FCOS)
3. 动态分配 (如ATSS)
4. 基于质量的分配 (如OTA)
5. 自适应分配 (如AutoAssign)
// ... 其他策略 ...
